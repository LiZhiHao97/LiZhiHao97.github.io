    改过端口后 连接服务器会连不上

    出现
    : ssh: connect to host ip地址 port 22: Connection refused

    这时候就要虚入输入端口号了

    • ssh -p 8888 ubuntu@ip地址
      (8888是刚刚改过的端口)

    curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash

    装完后若发现nvm 不是一个命令 在来一个命令窗口

    安装好nvm 就可以安装nodejs了

    nvm install v8.2.1

    指定一下node版本

    nvm use v8.2.2

    设置一下默认

    nvm alias default v8.2.1
